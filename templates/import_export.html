Import/Export Instructions
Supported File Formats:
  • CSV: Comma-separated values (Recommended)
  • Excel: .xlsx or .xls files
  • JSON: JavaScript Object Notation
Requirements:
  • Maximum file size: 10MB
  • Required fields for books: Title, Author, Subject
  • Use UTF-8 encoding for CSV files
  • For Excel export: Install openpyxl package
Tips:
  • Always download and use the template for imports
  • Backup your data before performing bulk operations
  • Test imports with a small file first
  • Check the import summary for any errors
